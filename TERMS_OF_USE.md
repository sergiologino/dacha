# Условия использования

Редакция от 15.03.2026.

Настоящие Условия регулируют использование сервиса «Любимая Дача» при входе по номеру телефона.

## Общие положения

- сервис предоставляет пользователю доступ к функциям приложения после успешной авторизации;
- вход по телефону подтверждается одноразовым SMS-кодом;
- пользователь обязан указывать номер телефона, к которому у него есть законный доступ.

## Правила авторизации

- запрещено запрашивать коды подтверждения для чужих номеров без разрешения владельца;
- запрещено передавать полученный одноразовый код третьим лицам;
- администрация сервиса вправе ограничить отправку кодов при подозрительной активности или злоупотреблениях.

## Ограничение ответственности

Сервис не несёт ответственность за невозможность входа, если она вызвана:

- действиями мобильного оператора;
- работой SMS-провайдера;
- неверно указанным номером телефона;
- действиями пользователя, приведшими к компрометации кода подтверждения.

## Изменение условий

Администрация сервиса вправе изменять настоящие Условия. Продолжение использования сервиса после публикации новой редакции означает согласие пользователя с обновлёнными условиями.
