# Политика конфиденциальности

Редакция от 15.03.2026.

Настоящая Политика конфиденциальности применяется к сервису «Любимая Дача» и объясняет, как обрабатываются данные при входе по номеру телефона.

## Какие данные используются

- номер телефона;
- технические данные авторизации: время запроса кода, факт подтверждения номера, служебные идентификаторы SMS-отправки;
- общие технические логи, необходимые для безопасности и стабильной работы сервиса.

## Для чего используется номер телефона

Номер телефона используется только для:

- отправки одноразового кода подтверждения;
- входа в аккаунт;
- защиты от повторных и мошеннических попыток авторизации.

## Что сервис не делает

- не использует номер телефона для рекламы;
- не публикует номер телефона в открытом доступе;
- не передаёт номер телефона иным пользователям сервиса.

## Техническая отправка SMS

Для доставки одноразового кода номер телефона технически передаётся SMS-провайдеру `SMS.ru`. Такая передача используется исключительно для отправки кода подтверждения и не является самостоятельной маркетинговой обработкой данных.

## Срок хранения

Одноразовые коды и служебные данные авторизации хранятся только в объёме и на срок, которые необходимы для подтверждения входа, предотвращения злоупотреблений и обеспечения безопасности сервиса.

## Защита данных

Сервис применяет разумные технические и организационные меры для защиты данных от утраты, неправомерного доступа, изменения и уничтожения.

## Изменения политики

Администрация сервиса вправе обновлять настоящую Политику. Актуальная версия всегда публикуется по адресу страницы, на которой размещён этот документ.
